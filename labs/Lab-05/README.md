# 🎵 Lab 05: Music Generation using RNN (LSTM)

Taken from MIT deep learning course

## 📌 Overview
This lab demonstrates the use of **Recurrent Neural Networks (RNNs)**, specifically **LSTM (Long Short-Term Memory)** networks, to generate music. The model is trained on Irish folk songs represented in **ABC notation** and learns sequential patterns to create new musical compositions.

---

## 🎯 Learning Objectives
By completing this lab, students will be able to:

- Understand sequence modeling using RNN/LSTM  
- Preprocess textual music data (ABC notation)
- Understand Vector Embedding
- Train deep learning models for generative tasks  
- Generate new sequences using trained models  
- Analyze and evaluate generated outputs

## 🛠️ Tasks

### 1. Data Preprocessing
- Load dataset  
- Convert text into character sequences  
- Create input-output pairs for training  

### 2. Model Development
- Build LSTM-based RNN model  
- Define loss function and optimizer  
- Train model on sequence data  

### 3. Music Generation
- Generate new sequences using trained model  
- Apply temperature sampling  
- Convert sequences into music format  

### 4. Evaluation
- Evaluate coherence of generated music  
- Analyze creativity and structure
